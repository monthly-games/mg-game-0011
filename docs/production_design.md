## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **꾸미기** | 나만의 요정 숲을 꾸미는 재미 | ★★★★★ |
| **힐링** | 아름다운 숲에서의 편안함 | ★★★★☆ |
| **수집** | 다양한 장식물과 요정 수집 | ★★★★☆ |
| **성장** | 숲이 점점 커지는 만족감 | ★★★☆☆ |

### 1.2 만족감 요소 (Satisfaction)
- **장식 배치**: 새 장식물 배치 시 숲이 아름다워짐
- **요정 등장**: 새로운 요정 해금
- **자원 수확**: 자동 생산된 자원 수확
- **지역 확장**: 새로운 숲 지역 해금

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 테마별 숲 완성
├── 요정 컬렉션
├── 퀘스트 완료
├── 친구 숲 방문 보너스
└── 시즌 이벤트 달성
```

### 1.4 사회적 요소 (Social)
- **숲 방문**: 친구 숲 방문 및 선물 교환
- **협동 퀘스트**: 함께 완료하는 특별 퀘스트
- **자랑하기**: 숲 스크린샷 공유
- **랭킹**: 숲 아름다움 평가 랭킹

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (30초~2분)
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [자원 수확] → [장식 구매] → [배치]              │
│       ↑                          ↓               │
│       └──── [자동 생산] ← [숲 관리] ←┘          │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **자원 수확** (5초): 자동 생산된 마나 수확
2. **장식 구매** (10~20초): 상점에서 장식물 구매
3. **배치** (30~60초): 숲에 장식물 배치/재배치
4. **숲 관리** (지속): 요정 돌보기, 이벤트 확인
5. **자동 생산** (방치): 배치된 장식물이 마나 생산

### 2.2 세션 루프 (5~10분)
```
[접속] → [방치 보상] → [자원 수확] → [퀘스트 확인] → [꾸미기]
                                                       ↓
                              [친구 방문] ← [새 장식 배치]
```

**세션 구성:**
- 방치 보상 수확: 30초
- 퀘스트 확인/완료: 1~2분
- 꾸미기: 2~5분
- 친구 방문: 1~2분

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [마나 축적] → [지역 확장] → [새 장식 해금]           │
│         ↓                              ↓             │
│  [요정 수집]                    [테마 숲 완성]        │
│         ↓                              ↓             │
│  [프레스티지 전환] ←────────────────────┘            │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[방치 보상 화면]
    ↓
[메인 숲 화면] ────→ [상점]
    │                [요정 도감]
    │                [퀘스트]
    │                [친구]
    │                [설정]
    ↓
[배치 모드] → [장식 선택] → [위치 지정] → [완료]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **프리미엄 장식** | 40% | 특별 장식물 판매 |
| **리워드 광고** | 30% | 생산 가속, 보상 2배 |
| **편의 기능** | 15% | 창고 확장, 자동 수확 |
| **시즌 패스** | 10% | 시즌 한정 장식 |
| **광고 제거** | 5% | 일회성 구매 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 기본 장식물로 숲 꾸미기 가능
├── 광고 시청으로 보너스 획득
├── 느리지만 완전한 진행
└── 시즌 무료 트랙 보상

과금 유저 혜택:
├── 광고 제거 ($2.99)
├── 프리미엄 장식 팩 ($1.99~$9.99)
├── 자동 수확 ($3.99)
├── 창고 확장 ($1.99~$4.99)
└── 시즌 패스 ($4.99/시즌)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 보석 (100) | $0.99 | 장식 1~2개 |
| 중량 보석 (500) | $3.99 | 20% 보너스 |
| 대량 보석 (1200) | $7.99 | 40% 보너스 |
| 광고 제거 | $2.99 | 영구 적용 |
| 테마 팩 | $4.99~9.99 | 테마별 장식 세트 |
| 시즌 패스 | $4.99/시즌 | 시즌 한정 장식 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 출석 체크 (7일 사이클, 특별 장식 포함)
- **일일 퀘스트**: 수확, 배치, 친구 방문
- **주간 이벤트**: 테마 꾸미기 이벤트
- **시즌 패스**: 60일 시즌 보상 트랙
- **친구 선물**: 매일 친구에게 선물 가능

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
숲 규모
  ▲
  │                              ╭──────
  │                         ╭────╯
  │                    ╭────╯
  │               ╭────╯
  │          ╭────╯
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 게임 진행
    1    7   14   21   28   35  42일
```

**성장 공식:**
```
숲_레벨 = floor(장식물_수 / 10)
마나_생산 = 장식물별_생산량 합계
확장_비용 = 기본_비용 × (1.5 ^ 확장_횟수)
```

### 4.2 콘텐츠 진행
| 단계 | 해금 조건 | 새로운 요소 |
|------|----------|-------------|
| 작은 숲 | 시작 | 기본 장식 10종, 요정 1종 |
| 성장하는 숲 | 레벨 5 | 중급 장식, 요정 2종 |
| 아름다운 숲 | 레벨 10 | 새 지역 해금, 테마 장식 |
| 마법의 숲 | 레벨 20 | 희귀 장식, 특수 요정 |
| 전설의 숲 | 레벨 30 | 전설 장식, 무한 확장 |
| 신비의 숲 | 프레스티지 | 영구 보너스, 특수 장식 |

### 4.3 밸런싱 공식
```dart
// 마나 생산량 계산
int calculateManaProduction(Forest forest) {
  int baseMana = forest.decorations.map((d) => d.manaPerHour).sum;
  double bonusMultiplier = 1 + forest.bonuses.sum;
  return (baseMana * bonusMultiplier).toInt();
}

// 장식물 가격
int decorationPrice(Decoration deco) {
  int base = deco.tier * 100;
  double rarityMult = getRarityMultiplier(deco.rarity);
  return (base * rarityMult).toInt();
}

// 방치 수익 계산
int idleManaReward(Forest forest, Duration offlineTime) {
  int manaPerHour = calculateManaProduction(forest);
  int maxHours = 8; // 최대 8시간
  int hours = min(offlineTime.inHours, maxHours);
  return manaPerHour * hours;
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 마나 생산 | 방치 | 시간당 생산량 |
| 친구 방문 보너스 | 친구 숲 방문 | 100~500 마나 |
| 퀘스트 보상 | 퀘스트 완료 | 마나/보석 |
| 일일 보너스 | 출석 | 50~500 마나 |
| 업적 보석 | 마일스톤 | 10~200 보석 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 현재 마나/보석
├── 수확 가능 알림
├── 퀘스트 진행
└── 설정 버튼

레벨 2 (상호작용 시):
├── 장식 상세 정보
├── 배치 모드 UI
├── 상점 인터페이스
└── 요정 정보

레벨 3 (팝업):
├── 새 장식 획득
├── 지역 확장 확인
├── 설정 메뉴
└── 친구 선물
```

### 5.2 네비게이션 흐름
```
[메인 숲]
    ├── [배치] → [장식 선택] → [위치 지정]
    ├── [상점] → [장식물/특별 상품]
    ├── [요정] → [요정 도감]
    ├── [퀘스트] → [일일/주간/업적]
    ├── [친구] → [친구 숲 방문/선물]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 마나 수확 | 마나 이펙트 | 수확음 | - |
| 장식 배치 | 배치 애니메이션 | 배치음 | 진동 (약) |
| 새 장식 | 반짝임 효과 | 획득음 | 진동 (중) |
| 지역 확장 | 확장 연출 | 팡파레 | 진동 (강) |
| 요정 등장 | 요정 애니메이션 | 요정 소리 | 진동 (중) |

### 5.4 접근성
- **간단한 터치**: 탭/드래그 기본 조작
- **확대/축소**: 핀치로 숲 확대/축소
- **폰트 크기**: 3단계 조절
- **색맹 모드**: 아이콘 패턴 추가
- **자동 수확**: 자동화 옵션
