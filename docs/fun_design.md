## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 숲/요정과 상호작용                                   │
│         ↓                                               │
│  2. 자원 수집 및 꾸미기/시설 배치                        │
│         ↓                                               │
│  3. 요정/환경 성장/해금                                  │
│         ↓                                               │
│  4. 친구 숲 방문/가벼운 소셜 상호작용 ────→ (1로 반복)    │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **상호작용**: 요정들과 교감, 숲 환경 관리
2. **자원 수집/꾸미기**: 자원으로 숲 꾸미기, 시설 배치
3. **성장/해금**: 새로운 요정, 지역, 아이템 해금
4. **소셜**: 친구 숲 방문, 좋아요, 선물 교환

---

## 3. 재미 축 (Fun Pillars)

### 3.1 힐링 (Healing)
- **목표**: 마음이 편안해지는 경험 제공
- **핵심 경험**: 요정 숲에 들어오면 힐링되는 느낌
- **주요 요소**:
  - 아름다운 비주얼
  - 편안한 BGM/ASMR
  - 느긋한 페이스
  - 긍정적 피드백만 존재

### 3.2 꾸미기 (Expression)
- **목표**: 나만의 개성 있는 요정 숲 만들기
- **핵심 경험**: 내 숲에 대한 애착과 자부심
- **주요 요소**:
  - 자유로운 배치
  - 다양한 테마
  - 시즌 한정 아이템
  - 요정 외형 커스터마이징

### 3.3 소소한 상호작용 (Social Lite)
- **목표**: 부담 없는 가벼운 소셜 경험
- **핵심 경험**: 친구 숲을 구경하는 즐거움
- **주요 요소**:
  - 숲 방문 시스템
  - 좋아요/응원
  - 선물 교환
  - 공동 이벤트

## 4. 디자인 레버 (Design Levers)

### 4.1 감각 피드백
| 레버 | 설명 |
|------|------|
| 부드러운 피드백 | 모든 상호작용에 부드러운 애니메이션/사운드 피드백 제공 |
| 구현 방향 | 터치 시 반짝임 효과, 요정 반응 애니메이션, ASMR 사운드 |

### 4.2 환경 루틴 변화
| 레버 | 설명 |
|------|------|
| 시간/날씨 루틴 | 시간/날씨/이벤트에 따라 요정/환경 루틴 변화 |
| 구현 방향 | 아침엔 요정들 기지개, 비오면 버섯 아래 모임, 밤엔 반딧불 등장 |

### 4.3 테마 미션
| 레버 | 설명 |
|------|------|
| 목표 있는 꾸미기 | 테마별 꾸미기 세트 미션으로 목표 있는 꾸미기 제공 |
| 구현 방향 | 봄 꽃밭 테마 완성 → 특별 요정 해금, 할로윈 테마 → 한정 이펙트 |

### 4.4 소셜 보상 구조
| 레버 | 설명 |
|------|------|
| 가벼운 소셜 보상 | 방문/좋아요 기반 가벼운 소셜 보상 구조 설계 |
| 구현 방향 | 친구 방문 시 요정 기운 +10, 좋아요 받으면 꽃가루 획득 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 체류 시간 | 15분 이상 | 세션 시간 |
| 상호작용 횟수 | 일 30회 이상 | 터치 이벤트 |
| 꾸미기 변경률 | 주 5회 이상 | 배치 변경 로그 |
| 소셜 참여율 | 50% 이상 | 친구 방문 유저 비율 |
| D7 리텐션 | 40% 이상 | 7일차 잔존율 |
