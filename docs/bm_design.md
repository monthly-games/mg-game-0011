## 3. 상세 BM 설계

### 3.1 꾸미기/데코레이션 BM

#### 무료 구조
- 스토리 진행으로 기본 장식 40종 획득
- 시즌 이벤트에서 한정 데코 파밍
- 일일 미션으로 데코 코인 획득

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 숲 테마 세트 (장식 15종+) | $4.99~$9.99 |
| 단품 장식 | 개별 프리미엄 장식 | $0.99~$2.99 |
| 시즌 한정 세트 | 봄꽃/여름/가을/겨울 테마 | $6.99 |
| 숲 배경 | 전체 배경 변경 | $2.99~$4.99 |
| 날씨 효과 | 비/눈/벚꽃 등 | $1.99 |

### 3.2 요정 수집 BM

#### 무료 구조
- 스토리 진행으로 기본 요정 5종 합류
- 숲 가꾸기로 새 요정 방문 기회
- 이벤트에서 한정 요정 획득

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 요정 뽑기 | 10연차 1회 | 프리미엄 재화 1,800 |
| 픽업 배너 | 특정 요정 확률 UP | 동일 |
| 천장 시스템 | 40연차 SR 요정 확정 | - |
| 요정 의상 | 개별 코스튬 | $1.99~$3.99 |
| 요정 하우스 | 요정 전용 집 장식 | $2.99 |

### 3.3 방치 수익 BM

#### 무료 구조
- 오프라인 자원 수집 상한: 8시간
- 요정 활동 보상 누적

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 수집 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 수집 3배 패스 | 상시 적용 (30일) | $2.99/월 |
| 오프라인 확장 | 16시간까지 누적 | 패스 포함 |

### 3.4 소셜/방문 BM

#### 무료 구조
- 친구 숲 방문 무료
- 좋아요/응원 무료
- 일일 방문 보상

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 선물 패키지 | 친구에게 보내는 선물 | $0.99~$2.99 |
| 방문 이펙트 | 방문 시 특수 효과 | $1.99 |

### 3.5 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 자원 2배, 데코 코인, 요정 선물 |

## 6. BM 밸런스 원칙

### P2W 방지선
- 경쟁 요소 없음 (순수 힐링 게임)
- 요정 성능 차이 없음 (외형/개성만 차이)
- 꾸미기 아이템 기능 영향 없음

### 과금 압박 최소화
- 모든 요소 완전 선택적
- 시간이 지나면 대부분 콘텐츠 무료 접근
- 광고도 부담 없이 선택적
