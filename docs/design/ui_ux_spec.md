# MG-0011: Fairy Forest UI/UX 스펙

> **게임명**: Fairy Forest (요정의 숲)
> **장르**: 힐링 시뮬레이션
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 세로 정원 레이아웃
- 핀치 줌으로 디테일 감상
- 힐링 게임 세로 표준

---

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | 자원, 시간 |
| 하단 | 34-48dp | 도구, 메뉴 |
| 좌우 | 0dp | 숲 풀 너비 |

---

## 3. HUD 레이아웃

```
┌─────────────────────────────────────┐
│  [☀️ 오후 2시]  [🧚5]  [✨120] [⚙️] │  ← 상단 HUD
├─────────────────────────────────────┤
│                                     │
│    🌳      🌸      🌺              │
│         🧚                          │
│    🌷         🌻      🌳           │
│                                     │
│         🦋      🌹                  │  ← 숲 정원 (핀치 줌 가능)
│    🌼              🧚               │
│              🌳         🌸          │
│    🦌                               │
│         🌺      🌷                  │
│                                     │
├─────────────────────────────────────┤
│  [🌱심기] [💧물주기] [📷사진]       │  ← 도구 바
├─────────────────────────────────────┤
│  [정원][꾸미기][도감][상점]          │  ← 하단 네비게이션
└─────────────────────────────────────┘
```

### 3.1 상단 HUD

| 요소 | 크기 | 설명 |
|------|------|------|
| 시간 표시 | 100×32dp | 게임 내 시간 |
| 요정 수 | 60×32dp | 보유 요정 |
| 별가루 | 80×32dp | 화폐 |

### 3.2 정원 영역

| 요소 | 크기 | 설명 |
|------|------|------|
| 식물 | 48~96dp | 크기 다양 |
| 요정 | 32×32dp | 이동하는 요정 |
| 동물 | 40×40dp | 방문하는 동물 |

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 식물 선택 | 식물 위 | 탭 |
| 줌 인/아웃 | 정원 전체 | 핀치 |
| 이동 | 정원 전체 | 드래그 |
| 요정 터치 | 요정 위 | 탭 (보상) |
| 물주기 | 식물 위 | 탭 (도구 선택 후) |

### 터치 피드백
- 식물 탭: 반짝임 이펙트
- 요정 탭: 별가루 파티클
- 물주기: 물방울 애니메이션
- 줌: 부드러운 스케일 전환
