## 2. 핵심 KPI

| KPI | 목표 | 측정 방법 |
|-----|------|----------|
| 숲 꾸미기 변화 빈도 | 주 3회+ | 꾸미기 로그 |
| 방문/좋아요 수 | 일일 5건+ | 소셜 로그 |
| 세션당 체류 시간 | 8분 (짧지만 편안) | Firebase Analytics |
| D7 Retention | 30% 이상 | 일별 유저 추적 |
| 소셜 상호작용 비율 | 50%+ | 상호작용 로그 |

## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 꾸미기 세트/요정 캐릭터 배포 |
| 7~10주 | 소셜 기능 최소 단위 (방문/좋아요) 도입 |
| 11~12주 | 봄 테마 준비 |

**핵심 과제:**
- 꾸미기 시스템 안정화
- 소셜 기능 기초 구축
- 요정 캐릭터 매력 어필

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 봄 테마 숲 업데이트 |
| 5개월 | 소규모 협동 이벤트 (서버 목표) |
| 6개월 | 여름 테마 숲 업데이트 |
| 7개월 | 가을 테마 숲 업데이트 |
| 8개월 | 협동 이벤트 2차 |

**핵심 과제:**
- 계절/테마 숲 사이클 정착
- 협동 이벤트 효과 검증
- 소셜 기능 확장

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | IP/브랜드 콜라보 (테마 숲/요정 코스튬) |
| 10개월 | 0024 페스티벌 연계 준비 |
| 11개월 | 겨울 테마 숲 + 힐링 페스티벌 |
| 12개월 | 1주년 이벤트 + 연말 테마 |

**핵심 과제:**
- 브랜드 콜라보 검증
- 크로스 게임 연계
- 장기 유저 리텐션
